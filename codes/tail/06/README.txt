4. `-f` Option in Tail Command in Linux
This option is mainly used by system administration to monitor the growth of the log files 
written by many Unix program as they are running. 
This option shows the last ten lines of a file and will update when new lines are added. 
As new lines are written to the log, the console will update with the new lines.
