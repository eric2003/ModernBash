4. -nr option: To sort a file with numeric data in reverse order we can use the combination of two options as stated below. 

Example: The numeric file is the same as above. 

Syntax : 

$ sort -nr filename.txt
Command :
$ sort -nr file1.txt
Output :
200
89
50
39
15
