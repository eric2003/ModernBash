5.2 Bash Variables

BASH
The full pathname used to execute the current instance of Bash.

eric@DELL MINGW64 /d/work/bash_work/ModernBash/codes/bash-variables/BASH/01 (main)
$ bash testprj.sh
$BASH: /usr/bin/bash

eric@eric-virtual-machine:~/work/bash_work/ModernBash/codes/bash-variables/BASH/01$ bash testprj.sh 
$BASH: /usr/bin/bash
